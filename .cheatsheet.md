# Terminal Cheatsheet

## fzf — fuzzy finder
  ctrl+r        Fuzzy search shell history
  ctrl+t        Fuzzy find file, insert path at cursor
  alt+c         Fuzzy find directory and cd into it
  vim **<Tab>   Fuzzy completion for any command

## zoxide — smart cd
  z foo         cd to most frecent dir matching "foo"
  z foo bar     match multiple tokens
  zi            interactive picker (fzf) across known dirs
  z -           go back to previous dir

## delta — git diff pager (auto-activated)
  n / N         Jump to next/previous changed file
  q             Quit
  /pattern      Search within diff

## eza — better ls
  eza           like ls
  eza -la       long format with hidden files
  eza -la --git show git status per file
  eza --tree    tree view
  eza --tree --level=2

## ripgrep (rg) — fast search
  rg "pattern"          search recursively (respects .gitignore)
  rg "pattern" src/     specific directory
  rg -t ts "pattern"    only .ts files
  rg -l "pattern"       filenames only
  rg -i "pattern"       case insensitive
  rg "pattern" -C 3     3 lines context

## fd — fast find
  fd pattern        find files matching pattern
  fd -e ts          by extension
  fd -t d "src"     directories only
  fd --hidden pat   include hidden files

## bat — better cat
  bat file.ts       syntax-highlighted view
  bat -n file.ts    line numbers only
  bat --diff file   highlight changed lines
